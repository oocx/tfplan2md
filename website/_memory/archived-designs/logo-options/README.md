# tfplan2md Logo Design Options

This directory contains 15 logo design options for tfplan2md, created as part of the website redesign initiative.

## How to View

Open `compare.html` in a browser to see all logos side-by-side with:
- Multiple sizes (32px, 36px, 72px)
- Light and dark background previews
- Navbar context preview
- Interactive logo rotation in navbar

## Design Concepts

All logos are designed to:
- Work at small sizes (32-36px navbar scale)
- Support both light and dark themes
- Use the project's color scheme (brand blue #646cff)
- Be visually distinct from the old logo
- Use SVG format for scalability

## Technical Details

- **Format:** SVG (scalable vector graphics)
- **ViewBox:** 36×36 (1:1 aspect ratio)
- **Color Scheme:** 
  - Primary gradient: #646cff → #747bff
  - Accent colors: white for contrast
  - Internal accents: primary blue #646cff
- **Size Variants:** All logos work at 32px, 36px, and larger (tested up to 72px)
- **Theme Support:** Designed to work on both light and dark backgrounds

  - **Versatility** (works in multiple contexts)
