Oocx.TfPlan2Md: Transforming Terraform Plans to Markdown

What is Oocx.TfPlan2Md?

A tool to convert Terraform plan output into a human-readable Markdown report.

Key Features

  • Parses Terraform plan output
  • Maps Azure resources and principals
  • Generates Markdown reports
  • Supports diff highlighting
  • Customizable formatting

Architecture Overview

Components

  • Terraform Plan Parser: Parses the Terraform plan file.
  • Azure Resource Mapper: Maps resources to Azure entities.
  • Markdown Generator: Creates the Markdown report.
  • Scriban Helpers: Provides formatting and styling functions.

Scriban Helpers

Utilizes Scriban templating engine for dynamic report generation.
