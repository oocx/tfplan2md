The Problem

Before

  • No logical grouping
  • Cryptic JSON structure

  • Lost context

After

  • Structured sections
  • Module grouping
  • Readable markdown

  • Clear summaries

Install

$ docker pull oocx/tfplan2md:latest

Use

$ terraform show -json plan.tfplan | docker run -i oocx/tfplan2md

Output goes to stdout. Redirect to file or pipe to PR comment tool.

CI/CD

GitHub Actions
- name: Generate report
  run: |
    terraform show -json plan.tfplan | \
      docker run -i oocx/tfplan2md > plan.md

- name: Comment PR
  uses: actions/github-script@v6
  with:
    script: |
      const fs = require('fs');
      const body = fs.readFileSync('plan.md', 'utf8');
      github.rest.issues.createComment({
        issue_number: context.issue.number,
        owner: context.repo.owner,
        repo: context.repo.repo,
        body
      });
