This project uses comprehensive instruction files for AI agents. Read these before starting work:

1. **`.github/copilot-instructions.md`** - Generic AI agent guidelines (coding patterns, workflows, terminal commands)
2. **`docs/spec.md`** - Project specification, organization, coding standards, CI/CD
3. **`docs/agents.md`** - Agent-based workflow overview

- **`docs/architecture.md`** - Comprehensive arc42 architecture documentation
- **`docs/adr-*.md`** - Architecture decision records

### Development Guidelines
- **`docs/commenting-guidelines.md`** - XML documentation standards

- **`docs/features.md`** - User-facing feature documentation
